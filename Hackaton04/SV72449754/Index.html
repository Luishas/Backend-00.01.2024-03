¿Cómo defines una función?

Rpta: Para declarar una función, debe usar la palabra clave function (en-US), seguida del nombre de la función, el parámetro de la función, cerrados entre paréntesis () . Y, las declaraciones de JavaScript que definen la función, cerradas entre llaves {}.

Recordar que tambien se puede definir funciones flecha (()=>{}) y funciones anonimas,las cuales no se declaran como las mencionadas lineas arriba.

¿Hasta cuantos argumentos puedes declarar en una función?

Rpta: Segun la documentacion revisada una funcion en JavaScript admite 255 argumentos como maximo.
